# External Region Data Placeholder

This folder is reserved for region-specific raw data that will later complement the repository-derived references.

## Expected Data Categories

- livestock manure generation statistics
- municipal food or wet-waste generation statistics
- moisture and composition statistics for mixed waste
- regional electricity or fuel price data
- regional emission factors
- policy and subsidy reference documents

## Naming Suggestion

Use descriptive file names, for example:

- `livestock_manure_statistics_[region]_[year].xlsx`
- `food_waste_generation_[region]_[year].xlsx`
- `regional_energy_prices_[region]_[year].csv`
- `regional_emission_factors_[region]_[year].csv`

## Rule

When external regional data is added, always record:

- source organization
- source URL
- publication or download date
- whether the file is raw, manually exported, or pre-tabulated
